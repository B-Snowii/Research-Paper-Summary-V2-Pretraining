# 学术论文专业摘要 · V2 预训练

本仓库展示了工作流与实现，用于生成约 25 词学术摘要。

- 任务：每周批量生成 25 词摘要
- 核心成果：时间≤2 小时/周；方法/主题准确；风格一致；可复用流水线
- 背景与目标：4 天内完成定稿
- 现实挑战：数量多、主题/方法广、沟通成本高

## 版本说明（V2 · 预训练）
- 为何尝试：通过预训练注入领域知识，期望在方法/主题识别上超过纯 Prompt；探索权重初始化尺度。
- 为何放弃：高质量领域语料整理成本高且难全覆盖；工作论文新颖性强、同类材料稀缺，投入产出不佳。
